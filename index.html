Moviephile

A fun and interesting way to review movies

Moviephile is a Django based web application that I built that is used to review movies. The main difference is how the rating is determined. Users will score 10 different catagories on a scale out of 10 to determine the number of thumbs up a movie gets.

Python Web Scraper

A web scraper designed to scrape data from business entities in Colorado. This program takes data such as the business name, phone number and address and relates it to the business NAICS code, a piece of data that is notoriously hard to collect and expensive to purchase. This data is all stored in a SQL database and displayed by a GUI built using Tkinter. The user has the ability to search by any of the entity specific fields or browse businesses by the NAICS code.

Automatic Updating Service

WCF Web Service Built to Automatically Distribute Updates

This web service was built to allow customers to receive automatic updates. Previously, updates were distributed manually to customers costing many man hours each month and also causing delays in updates. Once this web service was completed, customer satisfaction increased immensely due to faster bug fixes. Employees were also much happier because they had to spend less time on the phone distributing updates.
